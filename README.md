
# AchadinhosWeb - Starter

Frontend prototype built with **Next.js** + **Tailwind CSS**.
Style: Dark Pinterest-style feed. Includes a simple `/admin` panel (uses localStorage) to add/edit/delete products.

## Quick start (in VS Code)

1. Extract the project and open it in VS Code.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```
4. Open http://localhost:3000

## Admin panel
Navigate to `/admin` to add, edit and remove products. Data is stored in your browser (localStorage).
